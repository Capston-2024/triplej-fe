# triplej-fe

Triple J 프론트엔드 레포지토리입니다.

## Description

`triplej-fe` 레포지토리는 아래와 같은 구조를 따릅니다. 
- `src/pages`
- `src/components`
- `src/assets`
- `src/constants`
- `src/styles`

주요 기능은 아래와 같습니다. 
- 로그인, 회원 가입
- 마이페이지 조회
- 채용공고 조회
- 채용 공고 지원

## How to Install
```bash
$ npm install
``원

## How to Run (Development)

```bash
$ npm start
```

위 명령어를 실행하면 `http://localhost:3000` 에서 개발중인 애플리케이션을 확인할 수 있습니다. 코드를 수정하면 자동으로 반영됩니다.

## How to Build

```bash
$ npm run build
```

`build` 디렉토리에 최적화된 프로덕션 빌드 파일이 생성됩니다.

## How to Test

```bash
$ npm test
```
